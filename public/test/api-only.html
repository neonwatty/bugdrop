Usage Example

This page uses data-button="false" to hide the floating button:

<script src="/widget.js"
        data-repo="owner/repo"
        data-button="false"></script>

Then trigger the modal from your own UI:

// Wait for BugDrop to be ready
window.addEventListener('bugdrop:ready', () => {
  document.getElementById('my-btn')
    .addEventListener('click', () => {
      window.BugDrop.open();
    });
});
