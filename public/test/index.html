How It Works

The feedback widget is loaded from /widget.js and creates an isolated environment using Shadow DOM. This ensures that:

  • Widget styles don't interfere with your page styles
  • Your page styles don't affect the widget appearance
  • The widget maintains consistent behavior across all sites
  • No CSS conflicts or naming collisions occur

When you submit feedback, it's sent to the /api/feedback endpoint as a JSON payload containing:

  • Title and description of the feedback
  • Optional screenshot (base64 encoded)
  • Optional annotations layer
  • Metadata (URL, viewport size, user agent, timestamp)
  • Optional element selector for targeted feedback
